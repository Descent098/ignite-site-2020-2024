Introduction to Python

In this session, we will get set up on our coding platform, take a look at what programming is, and begin writing our first lines of code! We will learn how to draw shapes, use variables, and add some interactivity to our program.

Session 2
Variables and If-Statements

Published: 07/13/2021

In this session, we will take a deeper look into using variables in our programs by exploring variable scope, combining variables, and a new data type: booleans! We will also learn about if-statements, and how we can use these in combination with booleans to apply some restraints to our code.

Session 3
If, Elif, Else statements and Logical Operators

Published: 07/20/2021

In this session, we will learn about how we can use elif and else statements in our if statements to add extra functionality to our code. We will then learn about logical operators, and how we can use these in our if statements. We will also begin to add some more interactivity to our programs by learning about keyboard and mouse input!

Session 4
Lists and For Loops

Published: 07/27/2021

In this session, we will learn about lists, and what makes them different than the variables we have previously learned about. We will then learn about adding loops to our code through “for loops”, and how we can use these loops in combination with lists to do some pretty cool stuff!

Session 5
While Loops and Functions

Published: 08/03/2021

In this session, we will learn about a new type of loop: a while loop! We will compare this loop with the previously learned “for loop”, and understand the key differences between them, and when we would want to use each one. Then, we will learn about functions: what they are, why and when we would use them, and how we can make our own!
